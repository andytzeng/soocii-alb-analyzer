# Statistic API calls within a time duration

## Command
```console
pipenv run python stat_api_calls.py 2018-03-28T19:50:00+8 2018-03-28T20:50:00+8
```

## Output
soocii-alb-analyzer/out/stats_2018-03-20T16:00:00_2018-03-22T15:59:59.csv
